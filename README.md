# 🔐 Password Manager

A simple and responsive **Password Manager** built with **React** and **Vite** that securely stores login credentials in the browser using **Local Storage**. Users can add, edit, delete, copy, and generate strong passwords without requiring a backend or database.

---

## 🚀 Features

* ✅ Add new website credentials
* ✏️ Edit existing passwords
* 🗑️ Delete saved passwords
* 📋 Copy username and password to clipboard
* 🔒 Generate strong random passwords
* 💾 Persistent storage using Local Storage
* 📱 Responsive and modern UI
* ⚡ Fast performance with React + Vite

---

## 🛠️ Tech Stack

* **React**
* **Vite**
* **JavaScript (ES6+)**
* **Tailwind CSS**
* **UUID**
* **generate-password-browser**
* **Local Storage API**

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/password_manager.git
```

Go to the project directory:

```bash
cd password_manager
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

## 💾 How It Works

All password data is stored locally in your browser using the **Local Storage API**.

The application allows you to:

* Save website credentials
* Update existing entries
* Delete saved passwords
* Generate secure passwords
* Copy credentials with a single click

Since there is **no backend**, your data remains on your own device and is not uploaded anywhere.

---

## ⚠️ Limitations

* Data is stored only in your browser.
* Clearing browser data will permanently remove all saved passwords.
* Passwords are **not encrypted**.
* Data cannot be synchronized across multiple devices.
* This project is intended for learning purposes and is **not recommended for storing sensitive passwords**.
